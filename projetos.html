GEL - Gestão e Estoque dos Laboratórios - Núcleo Integrado de Saúde

Esse projeto tem a finalidade de desenvolver um sistema que se chamará GEL – Gestão e Estoque dos Laboratórios – NIS, que será utilizado como proposta para o gerenciamento dos laboratórios. A linguagem adotada para o desenvolvimento será Ruby, que justamente com framework Rails, tem a intensão de demostrar a criação de uma aplicação web, para agilizar, facilitar o trabalho e criar um referencial teórico de desenvolvimento ágil e algumas tecnologias de programação. O Sistema guardará todas as informações em um banco de dados MySql, onde estará disponível para os gestores dos laboratórios e para os professores.

Equipe: Jean Dgardany Cavalcante de Lima

AGE - Ambiente para Gerenciamento de Eventos

Este projeto visa atender a uma demanda interna da instituição. O objetivo é proporcionar um ambiente para gerenciar eventos acadêmicos, organizando calendário e inscrições. O sistema é desenvolvido utilizando a tecnologia PHP.

Equipe: Ana Beatriz; Guilherme
